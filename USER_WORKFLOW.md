# User Workflow

1. Create or select active data pack.
2. Upload CSV files.
3. Review import errors.
4. Accept import.
5. Open Team page.
6. Review Niners top-5 pressure.
7. Review keeper/drop recommendations.
8. Open Trade Central.
9. Review players to shop and shield targets.
10. Open League Intel.
11. Review other teams' keeper pressure.
12. Open Draft Room.
13. Review pick values and projected pool.
