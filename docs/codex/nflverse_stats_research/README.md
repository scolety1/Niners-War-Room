# NFLVerse Stats Research Archive

This folder stores the deep-research reports used to move the War Room toward a stats-first model. Treat these files as research inputs, not executable implementation contracts, until contradictions and model-policy changes are audited.

## Archived Reports

| file | purpose |
|---|---|
| `2026-05-07_public_structured_data_stack.md` | Public/free structured data source stack for nflverse, ffverse, opportunity, identity, role, and local CSV import planning. |
| `2026-05-07_final_stats_first_veteran_model.md` | Proposed stats-first veteran model, feature hierarchy, normalization guidance, and LVE-specific formula recommendations. |
| `IMPLEMENTATION_REQUIREMENTS.md` | Extracted build requirements from the reports, separated from research prose and formula approval. |

## Review Rule

Do not mutate current decision rankings directly from these reports. First extract requirements, compare them against the current model, flag contradictions, then implement audited changes behind review-only gates.
