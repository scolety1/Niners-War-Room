# Source Firewall Data Stack Plan

This plan converts the 2026 data-source research into a Model v4 implementation
lane. It does not change formulas, rankings, My Team, War Board, readiness gates,
or recommendations.

## Core Decision

Private model value may read only admitted factual or transparent derived evidence.
Market, ADP, projection, mock, big-board, consensus, and trade-value feeds remain
display-only unless a future audited contract explicitly reclassifies a factual
field.

## New Registry

Machine-readable source policy starts here:

- `config/source_registry.csv`

The registry separates:

- `ADMITTED_FACT`
- `DERIVED_ADMITTED`
- `DISPLAY_MARKET`
- `DISPLAY_PROJECTION`
- `DISPLAY_OUTCOME_CONTEXT`
- `MANUAL_SCOUT`
- `REJECTED`

The first firewall test proves no row marked display-only or market/projection
context is allowed to drive private value.

## Recommended Stack

Private value:

- nflverse/nflreadr/nflfastR factual NFL evidence
- Sleeper league state, roster state, scoring settings, picks, and IDs
- actual completed NFL draft capital
- factual college/profile data with transparent formulas
- RotoWire factual/user-export tables only when classified as admitted evidence

Display-only:

- RotoWire projections, ADP, ranks, mocks, narrative blurbs
- FantasyPros ECR, ADP, projections, ranks
- DynastyProcess values/ECR
- PlayerProfiler ADP, rankings, trade values, big boards, mocks
- Tankathon mocks and big boards
- PFF grades/ranks/big boards unless explicitly reclassified later

## Implementation Order

1. Source registry and firewall tests.
2. Sleeper sync contract for league truth and player IDs.
3. nflverse ingestion contract for factual production, first downs, snaps, injuries,
   depth charts, draft picks, and combine.
4. RotoWire export importer that accepts only user-provided or explicitly
   permitted structured exports.
5. Missing evidence report.
6. Rookie draft-capital evidence table.
7. UI/export grouping labels for admitted evidence, display-only market context,
   projection context, source risk, and review-only safety.

## Non-Negotiables

- Missing evidence remains missing.
- Display-only context cannot become private value.
- Historical outcome buckets and similarity comps cannot feed ranking formulas.
- 10-team, 1QB, non-PPR, no-TE-premium settings must be stamped on review exports.
- No final cut, keep, trade, or draft recommendation is generated by this work.
