# Model Spec

V1 is deterministic and explainable.

Do not use generated AI text to make decisions. Model outputs should come from formulas:

- Pick value curve on a 1,000-point local scale.
- Position-specific private scores.
- Confidence score.
- Keeper score.
- Drop candidate score.
- Keeper pressure.
- Trade scores.

Official Rank controls league rules. It is separate from private value, market value, and War Score.
