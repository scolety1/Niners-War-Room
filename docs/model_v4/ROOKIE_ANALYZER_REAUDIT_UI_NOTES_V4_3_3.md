# Model v4.3.3 Rookie Analyzer UI Notes

The Draft Room / Prospect Board was upgraded after the rookie formula balance repair.

## Board Columns

The prospect board now exposes:

- overall rookie rank
- player, position, NFL team, and college
- final score
- production score
- College Team Share score
- NFL Draft Pick Signal
- athletic score
- age score
- confidence cap
- evidence availability
- Trust Level
- model edge / source warning label
- short "why this rank" text

## Filters

The UI includes filters for:

- pick window
- position
- board band
- Trust Level
- warning type
- NFL team
- draft round, including No draft capital

## Detail Drilldown

Each selected player has a drilldown area with:

- component breakdown
- receipts
- warnings
- NFL Draft Pick Signal rows
- workout profile
- depth chart context
- market context in a separate non-scoring tab

Market, ADP, rankings, and projection context are labeled as non-scoring.

## Label Changes

- Market Share is shown as College Team Share.
- Draft Capital is shown as NFL Draft Pick Signal.
- Evidence Status is shown as Trust Level.
- Manual Scout Source Review is shown as Needs Human Review.

## Guardrails

This UI change did not create final recommendations and did not mutate active rankings, My Team, War Board, readiness gates, or app promotion state.
