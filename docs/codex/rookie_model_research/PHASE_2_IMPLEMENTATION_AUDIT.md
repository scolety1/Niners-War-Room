## Cross-position integration rules

A deterministic local-first implementation should separate **pre-draft translation** from **post-draft utility**.

The safest architecture for LVE is:

`base_translation_score`  
Built only from pre-draft evidence-backed and stronger secondary features:
- QB: capital, rushing, passing efficiency, P2S/process.
- RB: capital, age-adjusted team-play production, receiving role, size-speed, efficiency composite.
- WR: capital, adjusted RYPTPA, breakout/declare timing, share-based production, YPRR.
- TE: capital, adjusted RYPTPA, receiving role, broad/speed, day-3 suppression.

`post_draft_overlay`  
Built only after the NFL Draft:
- landing spot,
- path to snaps/routes/touches,
- line/QB environment,
- return role,
- LVE-specific first-down and goal-line projection.

`confidence_score`  
Start at 100 and reduce it by:
- each rowŌĆÖs `missing_data_penalty`,
- hard risk flags,
- single-season spikes,
- provider-specific missingness,
- transfer-heavy or role-unstable profiles.

`lve_decision_score`  
This should determine your offline draft ranking versus released veterans and free agents. A practical version is:

- **55% long-term dynasty translation**
- **25% Year 1 LVE utility**
- **20% trade insulation / market value**

For LVE, the biggest implementation mistake would be forcing every rookie selection to beat only other rookies. The right comparison set is:

**rookie score vs. released-veteran score vs. free-agent score**

That matters especially at:
- QB in a 1QB/10-team format,
- TE in a no-premium format,
- thin Day 3 RBs with ŌĆ£good landing spots,ŌĆØ
- and any rookie profile built mostly on PPR-era heuristics that do not carry cleanly into no-PPR plus first-down scoring.

The rookie should win the slot only if he clears at least one strong bar:

- clearly superior long-term translation profile,
- strongly insulated draft capital,
- unusually good LVE scoring fit,
- or meaningful expected trade-value appreciation.

## Open implementation cautions

The strongest evidence in the gathered research supports **draft capital, positional production/efficiency, TE receiving production plus explosion, and QB rushing**. The weaker or more mixed areas are **breakout age/early declare as standalone variables, WR target share as a direct ranker, FDPRR as a prospect coefficient, RB broken-tackle/contact stats as standalone inputs, and most landing-spot narratives**. That means the model should be opinionated in this exact way:

- **Be strict** on capital and core production.
- **Be modest** on athletic testing unless the position-specific evidence is better.
- **Be cautious** with landing spot.
- **Be aggressive** in comparing middling rookies against released veterans.
- **Never let no-PPR LVE scoring inherit full-PPR rookie logic without adjustment.**
